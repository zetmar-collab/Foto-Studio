Foto-Studio 2.5
Autor: Marek Zettel

ZAWARTOSC PACZKI
----------------
WINDOWS
  Gotowa wersja dla Windows z plikiem Foto-Studio.exe.

MACOS
  Wersja dla macOS uruchamiana przez Python 3.

IKONY
  Ikony aplikacji w formatach PNG i ICO.

WINDOWS - URUCHOMIENIE
----------------------
1. Otworz folder WINDOWS.
2. Uruchom Instaluj-Foto-Studio-Windows.bat, aby utworzyc skrot i wystartowac aplikacje.
   Mozesz tez uruchomic tylko Uruchom-Foto-Studio.bat albo Foto-Studio\Foto-Studio.exe.
3. Aplikacja otworzy przegladarke pod adresem:
   http://localhost:3000

WINDOWS - SKROT NA PULPICIE
---------------------------
1. Otworz folder WINDOWS.
2. Uruchom Utworz-skrot-na-pulpicie-Windows.bat.
3. Na pulpicie pojawi sie skrot Foto-Studio.

MACOS - URUCHOMIENIE
--------------------
1. Otworz Terminal.
2. Przejdz do folderu MACOS.
3. Uruchom:
   chmod +x Instaluj-Foto-Studio-macOS.command install-macos.sh Uruchom-Foto-Studio.command Utworz-skrot-na-pulpicie-macOS.command
   ./Instaluj-Foto-Studio-macOS.command
4. Potem uruchamiaj aplikacje przez:
   ./Uruchom-Foto-Studio.command

MACOS - SKROT NA PULPICIE
-------------------------
1. Otworz Terminal w folderze MACOS.
2. Uruchom:
   ./Utworz-skrot-na-pulpicie-macOS.command
3. Na pulpicie pojawi sie plik Uruchom Foto-Studio.command.

WYMAGANIA
---------
Windows:
  Brak dodatkowych wymagan dla gotowego EXE.

macOS:
  Python 3.10 lub nowszy.

UWAGI
-----
Program dziala lokalnie na komputerze i zapisuje dane w przegladarce oraz w folderze kopii zapasowych wskazanym przez uzytkownika.
