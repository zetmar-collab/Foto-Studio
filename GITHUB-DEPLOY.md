# Foto-Studio 2.5 - publikacja na GitHub

Docelowe repozytorium:

`https://github.com/zetmar-collab/Foto-Studio.git`

Jeśli repozytorium jeszcze nie istnieje, utwórz je na GitHub jako:

`zetmar-collab/Foto-Studio`

Potem w tym folderze uruchom:

```powershell
git push -u origin main
```

Gotowa paczka ZIP aplikacji znajduje się w:

`release/Foto-Studio-2.5.zip`
